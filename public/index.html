Content Management System

Welcome! to this content management system developed by Group 7. The content on the webiste is fetched using a RSS API. NoSQL database is used to store and manage the content displayed upon the sceen.
MongoDB is used for the management of the NoSQL database.
This website has 2 different pages. One is the Homepage, where all the content can be seen. Other, is the query page, where user can add different queies regarding database.
